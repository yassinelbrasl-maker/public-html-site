Acomptes provisionnels IRPP
Chaque acompte = 30% de l'IRPP N-1. Personnes physiques : 25 juin · 25 sept. · 25 déc.
Rappel : Les acomptes versés sont déduits de l'IRPP final lors de la déclaration annuelle (avant le 25 mai de l'année suivante pour les professions libérales).
Retenues à la source (RAS)
En tant qu'architecte (personne physique), vous subissez une RAS de 10% sur vos honoraires lorsque le client est une entreprise ou une administration. Cette retenue est une avance sur votre IRPP — vous la récupérez via la déclaration annuelle.
Obligatoire ? Oui, pour tout client entreprise/administration. Ils doivent remettre un certificat RAS via la plateforme TEJ (DGI). Pour les particuliers : pas de RAS.

Guide — Activer WebDAV sur votre QNAP
1. Ouvrez l'interface QTS → App Center → recherchez et installez WebDAV
2. Lancez WebDAV → activez le service → notez le port (défaut HTTP : 5005)
3. Dans Panneau de configuration → Dossiers partagés → clic droit sur votre dossier → Modifier les autorisations → autorisez votre compte en lecture
4. Renseignez le port WebDAV et le nom du dossier ci-dessus
5. Test : ouvrez http://[IP_NAS]:5005/dav/[dossier] dans votre navigateur
Guide — Créer une clé API myQNAPcloud
1. Connectez-vous sur account.qnap.com avec votre compte QNAP
2. Allez dans Security → Application Credentials (ou API Keys)
3. Cliquez Create Application → donnez le nom "Cortoba Atelier"
4. Copiez la clé générée dans le champ "Clé API myQNAPcloud" ci-dessus
5. Cette clé permet de récupérer automatiquement la capacité et l'état du NAS depuis internet
